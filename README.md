# Pipes

Pipes are a collection of `HTTP::Handler`'s which extend the possibilities of the Grip framework.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     pipes:
       github: grip-framework/pipes
   ```

2. Run `shards install`

## Usage

```crystal
require "pipes"
```
